# 동의서 공유 - 에듀테크 개인정보 동의서 플랫폼

전국 선생님들이 에듀테크 개인정보 동의서를 "한 번 만들면 모두가 쓰는" 공유 플랫폼입니다.

## 기능

- **검색/다운로드**: 서비스명으로 검색하여 hwpx 동의서 파일 다운로드
- **합본 다운로드**: 여러 서비스를 선택하여 하나의 hwpx 파일로 합본
- **자동 제작**: 이용약관 텍스트 또는 URL을 입력하면 AI가 분석하여 동의서 자동 생성

## 기술 스택

- **Frontend**: Next.js + React + Tailwind CSS
- **Backend**: Vercel Serverless Functions
- **Database**: Supabase (PostgreSQL)
- **Auth**: 카카오 로그인 (Supabase Auth)
- **AI**: Claude API (약관 분석)
